Bien démarrer

Cet outil rend vos images beaucoup plus légères pour le web, en 3 étapes. Aucune image ne quitte votre appareil — tout se passe dans le navigateur, même sans connexion.

  1. 1
    Ajoutez vos images
    Glissez-les dans la zone à gauche, ou cliquez pour parcourir. JPG, PNG, GIF, BMP, AVIF, SVG, ICO… plusieurs à la fois.
  2. 2
    Gardez les réglages par défaut
    Rien à comprendre pour commencer. Recette conseillée : Qualité 75. Pour de très grandes photos, activez le Redimensionnement et choisissez un préréglage.
  3. 3
    Convertir, puis Télécharger tout
    Le bouton Convertir est en bas. Récupérez ensuite vos fichiers (un ZIP si plusieurs) via Télécharger tout.
Besoin de recadrer pour Instagram, Facebook ou LinkedIn ? Bouton Recadrer sur chaque image, avec les formats prêts à l'emploi.

Garder WebP Forge à portée de main

WebP Forge est un fichier autonome : rien à installer, aucun compte. Pour le rouvrir en un clic, épinglez-le une fois.

  • 1
    Ajouter aux favoris / à la barre de lancement
    Appuyez sur Ctrl + D, confirmez, puis glissez le favori dans votre barre de favoris. C'est la façon la plus rapide de relancer l'outil.
  • 2
    Créer un raccourci « application » (fenêtre dédiée)
    Sur Chrome / Edge : menu ⋮ → Enregistrer et partager → Créer un raccourci…, puis cochez Ouvrir dans une fenêtre. Vous obtenez une icône et une fenêtre sans onglets, comme un vrai logiciel.
    Bonus : ce raccourci ouvre toujours dans Chrome — même si votre navigateur par défaut est un autre.
Dans tous les cas, WebP Forge fonctionne hors-ligne et vos images ne quittent jamais votre appareil.

À propos de WebP Forge

WebP Forge convertit et allège vos images pour le web (WebP), avec redimensionnement, recadrage et pack SEO. Tout se passe dans votre navigateur : aucune image n'est envoyée sur un serveur, même sans connexion.

  • ✓
    Formats acceptés : JPG · PNG · GIF · BMP · AVIF · SVG · ICO · WebP
  • ✓
    Formats de sortie : WebP · PNG · JPG · BMP · ICO — la conversion marche aussi dans l'autre sens (WebP → PNG, JPG…)
  • ✓
    100 % local · hors-ligne · sans compte ni installation
  • ✓
    Fonctionne sur Chrome, Edge, Firefox et Safari (encodeur WebP intégré)
